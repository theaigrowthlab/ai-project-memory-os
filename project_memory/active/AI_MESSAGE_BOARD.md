# AI Message Board

**Purpose:** Structured handoffs between AI roles. Organisational memory — survives chat loss and provider switches.

**Owner:** PM curates · All roles read/write

---

## Posting Convention

Append at the TOP (newest first):

```text
### [YYYY-MM-DD HH:MM] From: <Role> · To: <Role or ALL> · Type: handoff|blocker|decision-needed|recommendation|info
- Context: one line.
- Finding / recommendation: one line.
- Action requested + owner: one line.
- Links: file/commit/route if any.
- Status: open|in-progress|resolved|blocked
```

**Rules:**
- Evidence-based — link a file/commit/route
- `decision-needed` entries must also go in `DECISIONS_LOG.md` as **Proposed**
- No secrets, no unverified live claims
- PM archives resolved threads monthly
